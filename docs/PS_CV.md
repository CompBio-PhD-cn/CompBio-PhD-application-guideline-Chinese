# 文书

申请中另外一个非常重要的就是文书，通常来说包括 Personal Statement/Statement of Purpose,CV ，其中 CV 讲究的是在 1-2 页中展现申请人尽可能多的优点，并且涵盖重要的基本信息。而 PS 是串联过去的科研经历，并且展望 PhD 阶段想要做什么的关键材料。从 personal 就可以看出，PS是非常个人色彩的，每个人的 PS 都大不相同
